-   normalize (css)
-   tailwindcss

## Automation

Automation is built on the basis of bundler prettier and provides the following functionality.

-   live server
-   minification
-   add source maps
-   add prefixes
-   convert pug to html
-   add PWA and Service Worker

## Installation

```bash
npm install
```

## How to use

Development project:

```bash
npm run start:dev
```

Build a project for publication:

```bash
npm run build:prod
```

Clean:

```bash
npm run clean
```
